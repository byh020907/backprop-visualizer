# 순전파 · 역전파 · 가중치 업데이트 시각화

단일 HTML 파일로 동작하는 신경망 학습 시각화 도구입니다. Canvas 2D를 사용하며 외부 라이브러리가 필요 없습니다.

## URL
https://byh020907.github.io/backprop-visualizer/

## 사용법

`index.html`을 브라우저에서 열면 됩니다.

### 기본 조작
1. **설정 패널**(좌측)에서 네트워크 구조와 학습 파라미터를 조정
2. **초기화** 버튼으로 네트워크 생성
3. **순전파 → 역전파 → 업데이트** 순서로 한 단계씩 실행
4. **자동 실행**으로 전체 과정 반복

### 단축키
| 키 | 동작 |
|----|------|
| `I` | 초기화 |
| `F` | 순전파 |
| `B` | 역전파 |
| `U` | 업데이트 |
| `S` | 한 스텝 (순전파+역전파+업데이트) |
| `Space` | 자동 실행 |
| `R` | 리셋 |
| `Escape` | 뉴런 상세 닫기 |

### 뉴런 클릭
캔버스의 뉴런을 클릭하면 우측 패널에 상세 정보가 표시됩니다:
- ① 다음 층에서 전달된 오차 신호
- ② 이전 뉴런들의 영향 (가중치 테이블)
- ③ Bias(편향)
- ④ 출력 계산 과정

## 기능
- 네트워크 구조 설정: 입력층/은닉층(복수)/출력층 뉴런 수
- 활성화 함수: 은닉층(시그모이드 고정), 출력층(시그모이드/Linear 선택)
- 학습률, 난수 시드, 디스플레이 모드 설정
- 에지 굵기 배율 조정
- 한글 UI, 중학생도 이해할 수 있는 설명
